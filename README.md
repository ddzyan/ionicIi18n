## 简介
ionic3 国际化解决方案

### 使用的模块版本
1. "@ngx-translate/core": "^9.1.1",
2. "@ngx-translate/http-loader": "3.0.1",
3. "@angular/core": "5.2.11"

### 运行
```bash
git clone https://github.com/ddzyan/ionicIi18n.git

cd  ionicli18n

npm i

ionic serve
```

### 问题
#### v10.0.1 : TypeError: Object(...) is not a function
V10与Angular 5不兼容，请采用6+版本
